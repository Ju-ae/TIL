## 4. 추가 설명
<li> **clone**
: '복제'라는 의미로, repository를 복제한다는 의미로 사용된다.
새로운 local repository를 만들때 주로 사용될듯.
<li> **add**
 : sourcetree에 원래 사용하던 repository를 사용할 수 있도록 추가할 때 사용된다.
<li> **create**
:  로컬/git 에서 동시에 새로운 repository를 생성한다.
